# OpenCV_Project_GUI for Window

### A Graphic User Interface of my OpenCV-C++ project.
Use for Text Recognition (OCR), LICENSE PLATE DETECTION, Barcode and QR code detection
### Using OpenCV, Tesseract, Openalpr, and Zbar
### Requirements

  * [OPENCV](https://opencv.org/)
  * [Tesseract](https://github.com/tesseract-ocr/tesseract)
  * [Openalpr](https://github.com/openalpr/openalpr)
  * [Zbar](https://github.com/NaturalHistoryMuseum/ZBarWin64)
  * [Visual Studio](https://visualstudio.microsoft.com/downloads)
